\section{Project Definition}
% Task: Provide a clear description of the problem you plan to address.

LLMs are given more and more autonomy in society → Creates risks → We need trust. LLMs have been shown to lie. We can't trust models that lie. While honesty benchmarks exist \parencite{renMASKBenchmarkDisentangling2026}, there is currently no reliable way to measure and compare model honesty in more realistic multi-turn conversations.

\section{Motivation}
% Task: Briefly explain why this problem is worth addressing from both societal and scientific perspectives.
scientific
- Can be used to compare different honesty interventions → develop better ones → use in alignment audits...
- Can be used as a sign to model developers if something went wrong during training (Models show higher dishonesty)
- Can give insight into why models be

Societal
- Allows us to choose honest models
- Set an incentive for model provider to make their models more honest (what gets measured gets improved) → more honest models for society


\section{Background}
% Task: Summarize what is known in the scientific literature about this problem. 
% Base this on at least 5 relevant sources (last 5 years, peer-reviewed).
This project aims to investigate deceptive behavior in LLMs. Specifically, we aim to measure and compare their propensity to commit lies of commission in multi-turn conversations.
