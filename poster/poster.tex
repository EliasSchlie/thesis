\vspace{2mm}
\raggedright
{\Large\baselineskip=1.15\baselineskip\rmfamily
To what extent does \textbf{evolutionary optimization} outperform zero and multi-shot generation
in discovering \textbf{realistic scenarios} that induce \textbf{instrumental deception} in safety-aligned LLMs?\par}
\vspace{2mm}
}

\headerbox{Why It Matters}{name=why,column=0,span=2,below=rq}{
\vspace{3mm}
LLMs can \hl{strategically deceive} users when given misaligned incentives,
passing safety benchmarks while lying in deployment
\muted{\scriptsize\parencite{parkAIDeceptionSurvey2024a, greenblattAlignmentFakingLarge2024}}.

\vspace{2mm}

Moreover, models can detect when they are being tested
and \hl{fake compliance} \muted{\scriptsize\parencite{needhamLargeLanguageModels2025}},
so test scenarios must be realistic to produce valid results.

\vspace{3mm}

\callout{Finding \hl{realistic} deception scenarios currently requires
\hl{human experts} to manually craft and verify each test case,
making the process expensive and incomplete.}

\vspace{3mm}

Evolutionary methods may address both problems:
they can generate scenarios \hl{automatically} and \hl{adapt} as models improve,
potentially maintaining evaluative pressure where static benchmarks saturate
\muted{\scriptsize\parencite{ottMappingGlobalDynamics2022}}.
\vspace{3mm}
}

\headerbox{Key Concepts}{name=defs,column=0,span=2,below=why}{
\concept{\color{navy}Instrumental Deception}{%
Goal-directed lies of commission: strategically false statements
to achieve an objective. The model \emph{has} correct knowledge
but chooses to misrepresent it.
{\scriptsize\parencite{renMASKBenchmarkDisentangling2026, mahonDefinitionLyingDeception2016}}}

\hspace{4mm}\muted{\small$\neq$ Hallucination (lacks knowledge)
or Sycophancy (echoes user). Neither is goal-directed.}

\vspace{2mm}

\concept{\color{navy}Red-Teaming}{%
Adversarial testing to find inputs that cause undesired behavior;
here, scenarios that induce deception.
Automated red-teaming uses algorithms instead of human testers.}

\concept{\color{navy}Scenario}{%
A multi-turn context (system prompt, user messages, tool interactions)
designed to create an incentive for the target LLM to deceive.}

\concept{\color{navy}Evaluation Awareness}{%
LLMs can detect when they are being tested and strategically
fake alignment {\scriptsize\parencite{needhamLargeLanguageModels2025, greenblattAlignmentFakingLarge2024}}.
This motivates our realism constraint: scenarios must look
plausible to produce valid results.}
}

\headerbox{Related Work}{name=related,column=2,span=4,row=0}{
\begin{minipage}[t]{0.47\linewidth}
\textbf{\color{navy}Deception Benchmarks}\\[1mm]
MASK and DeceptionBench measure LLM dishonesty across societal domains,
but rely on \hl{manual scenario creation},
requiring human contributors to generate and verify each test case
\muted{\scriptsize\parencite{renMASKBenchmarkDisentangling2026, huangDeceptionBenchComprehensiveBenchmark2025}}.
\end{minipage}
\hfill
\begin{minipage}[t]{0.47\linewidth}
\textbf{\color{navy}Automated Red-Teaming}\\[1mm]
ForgeDAN, LLM-Virus, and Rainbow Teaming use evolutionary methods
to generate adversarial prompts, but target
\hl{generic policy violations}, not deception specifically.
\muted{\scriptsize\parencite{chengForgeDANEvolutionaryFramework2025, yuLLMVirusEvolutionaryJailbreak2024, samvelyanRainbowTeamingOpenEnded2024}}.
\end{minipage}

\vspace{2mm}
\callout{\hl{Gap:} No existing work applies evolutionary optimization
specifically to \hl{instrumental deception} with realism as a fitness constraint.}
}
